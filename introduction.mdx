Google Cloud Dataflow Templates are a collection of pre-implemented pipeline solutions for common in-Cloud data tasks: import, export, backup, restore, and bulk API operations. Built on [Apache Beam](https://beam.apache.org/), these templates run on the fully managed [Google Cloud Dataflow](https://cloud.google.com/dataflow/) service.

## What are Dataflow Templates?

Dataflow Templates let you run pre-built data pipelines without writing code. You provide parameters — source, destination, configuration — and Dataflow handles the rest. Templates come in two flavors:

- **Flex Templates** (v2): Packaged as Docker containers in Artifact Registry. Support dynamic parameters and advanced configurations. All new templates use this format.
- **Classic Templates** (v1): Staged as files in Cloud Storage. Simpler but more limited in parameter types.
